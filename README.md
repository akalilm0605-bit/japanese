# 日语单词拼写练习

一个简单的日语单词练习网页。看到中文释义后，输入对应的日语单词，网页会立即判断答案。

## 已有功能

- 随机显示中文释义
- 支持点击“提交答案”或按 Enter 提交
- 显示回答正确或错误
- 显示正确单词、假名和中文释义
- 显示正确数、错误数和正确率
- 内置 20 个常用日语单词
- 电脑和手机均可使用
- 相邻两题不会重复

答题区有“汉字”和“假名”两个输入框，中间的斜线已经固定。你只需分别填写 `学校` 和 `がっこう`，不需要自己输入斜线。两部分都正确才算回答正确。

## 最简单的预览方法

直接下载仓库后，双击 `index.html` 即可在浏览器中打开。

如果电脑已经安装 Node.js，也可以在项目文件夹中运行：

```bash
npm start
```

然后打开终端显示的网址。

## 检查项目

本项目没有需要额外安装的程序包。安装检查、自动测试和构建命令如下：

```bash
npm install
npm test
npm run build
```

构建完成的网页会放在 `dist` 文件夹中。

## 用 GitHub Pages 免费发布

1. 打开 GitHub 中的 `japanese` 仓库。
2. 点击仓库上方的 **Settings**。
3. 点击左侧的 **Pages**。
4. 在 **Build and deployment** 下，将 **Source** 选择为 **Deploy from a branch**。
5. 在 **Branch** 中选择 `main`，文件夹选择 `/ (root)`。
6. 点击 **Save**。
7. 等待几分钟，页面上会出现可以访问的网址。

以后只要更新 `main` 分支，GitHub Pages 就会自动更新网页。

## 文件说明

- `index.html`：网页内容
- `styles.css`：颜色、排版和手机适配
- `data.js`：20 个日语单词
- `app.js`：答题交互
- `logic.js`：答案判断和成绩计算
- `test/logic.test.js`：自动测试
- `scripts/build.mjs`：构建检查
